# FrameTee

FrameTee is a TAS editor for the game [DDNet](https://github.com/ddnet/ddnet). It's built using C99, Vulkan and ImGui.

## Features

- TAS editing
- Vulkan graphics
- ImGui user interface

## Requirements

- clang compiler
- Vulkan SDK
- zlib
